# n8n Workflows — справочные коллекции

Подборки готовых n8n-workflow для самостоятельного изучения. Не часть сдачи домашки M5 — reference: смотрите паттерны, ноды, структуру.

## Внешние репозитории (клонируйте напрямую)

| Репозиторий | Что внутри |
|---|---|
| [enescingoz/awesome-n8n-templates](https://github.com/enescingoz/awesome-n8n-templates) | Курируемый список n8n-шаблонов по категориям (AI-агенты, интеграции, автоматизация). |
| [Zie619/n8n-workflows](https://github.com/Zie619/n8n-workflows) | Большая коллекция готовых workflow-JSON с поиском и документацией. |

Чтобы получить их локально:

```bash
git clone https://github.com/enescingoz/awesome-n8n-templates.git
git clone https://github.com/Zie619/n8n-workflows.git
```

## Локальная подборка

| Папка | Что внутри |
|---|---|
| `n8n from Google Sheet/` | 313 workflow из community-таблицы (180 с загружаемым JSON, 133 reference-only), сгруппированы по авторам. Старт: `n8n from Google Sheet/INDEX.md`. |
