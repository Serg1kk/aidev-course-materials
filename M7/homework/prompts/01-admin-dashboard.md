# Готовый промпт: админ-дашборд трекинга (DZ1, шаг 5)

> Скопируй блок в кодинг-агента в корне форка. Дашборд читает коллекцию `chatlogs`, которую пишет
> роутер (см. [`02-n8n-router.md`](https://github.com/Serg1kk/aidev-course-materials/blob/main/M7/homework/prompts/02-n8n-router.md)). Логи смотрит АДМИН — это и есть «трекинг», который ты показываешь.

---

```
Ты работаешь в моём форке proshop_mern. Задача: добавить АДМИН-страницу «AI Router Dashboard», которая
показывает лог всех обращений к AI-ассистенту и решения роутера.

ШАГ 0 — карта форка (сделай сначала, кратко):
- где описаны admin-роуты бэка и middleware admin (доступ только админу);
- как устроены существующие admin-экраны фронта (например список пользователей/заказов) — повтори их паттерн;
- подключение к Mongo.

МОДЕЛЬ ДАННЫХ (создай, если ещё нет):
Коллекция chatlogs, поля: createdAt, userId, message, piiEntities (массив строк, напр. ["EMAIL_ADDRESS"]),
route ("local"|"cloud"), model (строка), reply (строка), latencyMs (число), costUsd (число).
ВАЖНО: записывать в эту коллекцию будет роутер (n8n или код роутера). Здесь нужно только ЧИТАТЬ.

ЗАДАЧА:
1. Бэк: GET /api/assistant/logs — ТОЛЬКО для админа (используй существующий admin-middleware),
   возвращает последние N записей chatlogs (сортировка по createdAt desc, пагинация по желанию).
2. Фронт: admin-страница AIRouterDashboardScreen — таблица колонок:
   время | userId | сообщение | найденный PII | маршрут (local/cloud) | модель | ответ | латентность | $.
   Приватные (local) запросы показывай как $0.00. Подсветь строки local иначе, чем cloud.
3. Простая сводка сверху: всего запросов, сколько ушло в local vs cloud, суммарная экономия $.
4. Добавь ссылку на страницу в админ-навигацию (рядом с Users/Orders/Products).

ОГРАНИЧЕНИЯ:
- Доступ строго админский (проверка на бэке, не только скрытием в UI).
- Стиль — как у существующих admin-экранов. Минимальные изменения.

КРИТЕРИЙ ГОТОВО:
- Зашёл админом → открыл AI Router Dashboard → вижу строки по каждому сообщению юзеров с маршрутом и
  моделью; не-админ страницу не открывает (403/редирект).

Покажи созданные/изменённые файлы и как открыть страницу.
```
